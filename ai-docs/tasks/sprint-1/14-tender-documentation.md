# 14 — Тендерная документация

**Приоритет:** P1  
**Оценка:** 2–3 дня  
**Зависимости:** задачи 01–09 (для актуальных скриншотов и матрицы)

## Описание

Подготовить комплект документов для участия в тендере на корпоративный сайт медицинской клиники. Акцент: **сайт клиники + модуль онлайн-заказа анализов** как комплексное решение.

## Ограничение

В документации явно разделять два модуля:

1. **Сайт клиники** — врачи, услуги, запись, контакты, информационные разделы
2. **Модуль диагностики** — каталог анализов, корзина, оформление, личный кабинет заказов

Не позиционировать проект только как «лабораторный магазин».

Создать в `ai-docs/tender/`:

## Demo-сценарии (для видео)

1. **Пациент записывается к врачу:** главная → врачи → карточка → запись → подтверждение → ЛК
2. **Пациент заказывает анализы:** каталог → корзина → дом/клиника → оформление → заказ в ЛК
3. **Новый пользователь:** регистрация → согласие ПДн → профиль
4. **Информационный раздел:** для клиентов → FAQ → ссылка на анализы
